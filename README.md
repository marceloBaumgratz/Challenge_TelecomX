# Telecom X - Churn Analysis

Projeto de análise exploratória de dados (EDA) focado na evasão de clientes (Churn) da empresa Telecom X.

## Objetivo
Identificar padrões associados ao churn e gerar insights para retenção de clientes.

## Estrutura do Projeto

data/raw -> Dados brutos  
data/processed -> Dados tratados  
notebooks -> Análises em Jupyter/Colab  
src -> Scripts auxiliares  

## Tecnologias
- Python
- Pandas
- Matplotlib
